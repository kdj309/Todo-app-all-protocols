
# To-do application in most common protocols

## Application set-up
## HTTP 1.2
- server
  ```
    node index.js
  ```
To deploy this project run
- client 
  ```
    nginx
  ```
## gRPC
  - server
      ```
      npm init -y
      npm install
      node index.js
      ```
  - client 
      ```
      npm install
      npm start
      ```
